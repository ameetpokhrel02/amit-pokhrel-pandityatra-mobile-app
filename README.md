# Pandit Yatra App

Amit Pokhrel's Pandit Yatra App is a web application built primarily with **TypeScript** and **JavaScript**. The app aims to connect users with pandits (Hindu priests) for religious ceremonies, yatra (pilgrimages), and puja bookings.

## Features

- **Pandit search & booking:** Find qualified pandits based on location, expertise, and availability.
- **Yatra management:** Browse and manage various religious pilgrimages and events.
- **Puja customization:** Choose from a range of pujas and rituals, with personalized booking options.
- **Profile system:** Both users and pandits can manage their profiles, bookings, and schedules.
- **Secure payments:** Integrated payment options to make booking easy and reliable.
